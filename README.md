# elm-mcp

An MCP server for agentic development of Elm applications.

#### Implemented tools
- `validate`: compiles project and identifies build errors
- `add_package`
- `remove_package`
- `search_packages`
- `get_latest_package_version`
- `get_docs`

#### How to use

- `PORT=1234 PROJECT_FOLDER=/path/to/elm/app cargo run`
